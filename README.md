# TextBox

## Um editor de texto simples em Java que atualmente conta com:

- Contador de linhas e caracteres
- Corretor automático
- Além de TXT, possui compatibilidade com o formato RTF para permitir o uso de diferentes fontes e tamanhos no mesmo texto
- Possui seletor de temas claro/escuro

## Adições futuras

- Compatibilidade com o formato markdown
